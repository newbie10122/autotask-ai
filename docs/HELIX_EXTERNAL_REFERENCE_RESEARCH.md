# Helix External Reference Research

## Purpose

When Helix manages AutoTask AI, Helix and her agents should use approved external repositories as reference material for AI workspace design, task workflow UX, dashboards, and controlled automation planning.

This file is a repo-level research queue. It does not approve code copying, dependency adoption, or implementation.

Useful for generated onboarding videos, task walkthroughs, and support/demo clips.
